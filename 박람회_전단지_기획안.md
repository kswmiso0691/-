# 치과 박람회 전단지/리플렛 기획안

## 핵심 컨셉

**"재료비 30% 절감, 믿기 힘드시죠?"**

박람회장에서 가장 중요한 건 **3초 안에 시선을 잡는 것**.

---

## 1. 헤드카피 (전면)

**옵션 A:** "원장님, 재료비 30% 아직도 안 줄이셨어요?"
**옵션 B:** "임플란트 재료비, 매달 얼마나 새고 있나요?"
**옵션 C:** "같은 퀄리티, 30% 더 싸게 — 비결은 단품주문"

> 숫자("30%")를 크게 강조하면 시선이 자동으로 멈춘다.

---

## 2. 이벤트 구성 (박람회 한정)

| 구분 | 내용 | 표현 방식 |
|------|------|-----------|
| **메인 이벤트** | 커스텀 어버트먼트 **50% 할인** | 가격 취소선 + 빨간 할인가 (시각적 충격) |
| **서브 혜택** | 임플란트 단품주문 시스템 소개 | "필요한 것만, 필요한 만큼" |
| **한정 조건** | 박람회 현장 상담 신청자 한정 | "선착순 OO명" 또는 "박람회 기간 한정" |

---

## 3. 전단지 레이아웃 구성

### 앞면 (3초 승부)
```
┌─────────────────────────────────┐
│  [충격적 숫자]                    │
│  재료비 30% 절감                  │
│  ━━━━━━━━━━━━━━━━━━━            │
│                                  │
│  박람회 한정                      │
│  커스텀 어버트먼트 50% 할인        │
│                                  │
│  [QR코드]  지금 바로 상담신청 →    │
└─────────────────────────────────┘
```

### 뒷면 (신뢰 + 상세정보)
```
┌─────────────────────────────────┐
│  왜 30% 절감이 가능한가?          │
│                                  │
│  1. 단품주문 = 불필요한 세트 X     │
│  2. 중간 유통 최소화              │
│  3. 커스텀 어버트먼트 자체 제작     │
│                                  │
│  [비교 표: 기존 vs 우리]          │
│  ─────────────────────           │
│  연락처 / 카카오톡 / QR코드       │
└─────────────────────────────────┘
```

## 4. 이목을 끄는 장치

### 시각적 장치
- **색상:** 빨간색 + 검정 조합 (할인 강조에 최적)
- **숫자 크기:** "30%"와 "50%"를 전체 면적의 1/3 이상 차지
- **취소선 가격:** ~~기존가~~ → **할인가** (가격 비교 즉시 인식)

### 행동 유도 장치
- **QR코드:** 스캔하면 바로 상담 신청 폼 연결
- **현장 한정 쿠폰:** 전단지 자체를 쿠폰으로 활용 ("이 전단지 지참 시 추가 혜택")
- **선착순 문구:** 긴급성 부여 → "박람회 기간 한정 선착순 50명"

### 박람회 현장 운영 아이디어
- **"재료비 절감 진단표"** — 간단한 체크리스트로 현재 지출 진단 → 자연스럽게 상담 유도
- **명함 드롭 이벤트** — 명함 넣으면 추첨으로 무료 샘플 제공 → 연락처 수집

---

## 5. 게릴라 쿠폰 배포 전략 (타 부스 슬쩍 배치)

### 쿠폰 디자인 포인트
- **크기:** 명함 사이즈 (90x50mm) — 작고 눈에 안 띄게 놓기 쉬움
- **디자인:** 전단지와 톤 다르게, 독립적으로 봐도 이해되는 구성
- **앞면:** "이 쿠폰 주우신 분께 — 커스텀 어버트먼트 50% 할인"
- **뒷면:** QR코드 + 간단한 연락처만

### 쿠폰 카피 예시
- "이걸 발견하셨다면, 운이 좋으신 겁니다"
- "히든 쿠폰 — 재료비 30% 절감의 비밀"
- "몰래 알려드립니다. 커스텀 어버트먼트 반값."
- "이 쿠폰은 아무에게나 드리지 않습니다"

### 배치 전략
- **타이밍:** 사람이 붐비는 시간대에 슬쩍 배치 (오전 개장 직후, 점심 직후)
- **위치:** 타 부스 카탈로그/리플렛 더미 사이, 휴게 공간 테이블 위, 세미나실 좌석 위
- **수량 조절:** 부스당 3~5장씩만 (너무 많으면 티남)
- **구별 표시:** 쿠폰에 고유번호 넣어서 어디서 수거됐는지 추적 가능

### 심리적 효과
- "우연히 발견한 것" → 호기심 + 특별함 느낌
- "히든 쿠폰" 프레이밍 → 남들은 모르는 혜택이라는 인식
- 작은 크기 → 부담 없이 주머니에 넣게 됨

### 주의사항
- 쿠폰에 회사명/연락처 반드시 포함 (출처 불명이면 역효과)
- 경쟁사 부스에 대놓고 쌓아두지 않기 (분쟁 방지)
- 자연스럽게 놓인 것처럼 보이도록 소량 배치
